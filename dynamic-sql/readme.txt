第一个入门的MyBatis例子

Mybatis实现步骤：
1. 新建的student表
2. 在pom.xml配置文件中加入maven的mybatis坐标，mysql驱动坐标
3. 创建实体类Student，这个实体类是保存数据库表中的一行数据。
4. 创建持久层的dao接口，定义操作数据库的方法。（这个方法名成用来设置mybatis配置文件中的id属性值）
5. 创建一个mybatis使用的配置文件
   这个配置文件叫做 ：sql映射文件，这个文件是用来写sql语句的。和接口中方法对应的sql语句。
   比如 ：增、删、改、查。
   一般一个表一个sql映射文件。这个文件是xml文件。
   要求：
        1. 这个XML文件写在接口所在的目录中。
        2. XML文件的名称和接口保持一致。

6. 创建mybatis的主配置文件：
   一个项目就一个主配置文件。
   主配置文件
        1. 提供了数据库的连接信息。（连接驱动、url、name、password）。
           需要把创建数据库的信息写在这个里边，数据库才可以使用。
        2. sql映射文件(使用compile编译后的执行sql的配置文件.xml)的位置信息。

7. 创建使用mybatis类，通过mybatis访问数据库。
